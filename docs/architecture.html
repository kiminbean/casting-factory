Implementation Roadmap

■ Phase 0 — 현재 구현 완료
UI 분리 완료 (v3.4.0): Next.js 16 관리자 웹 (생산 계획 / 주문 / 품질 / 입출고) + PyQt5 관제 모니터링 데스크톱 앱 (대시보드 / 생산 / 품질 / 물류), FastAPI + SQLite (개발용) + WebSocket, ESP32 컨베이어 v4.0 펌웨어 (TOF250 x2 + L298N + MQTT 지원)
■ Phase 1 — PostgreSQL 마이그레이션 + MQTT 파이프라인
SQLite → PostgreSQL 16 + TimescaleDB 마이그레이션 (다중 하드웨어 동시 쓰기 대응, JSONB, 시계열 하이퍼테이블), Alembic 스키마 관리 도입, RPi5에 Mosquitto 브로커 설치, FastAPI backend/app/mqtt/ 모듈 추가, ESP32 ↔ 서버 실시간 연동, WebSocket 이벤트 포워딩, Redis 캐시 추가 (pub/sub + 핫 데이터)
■ Phase 2 — Vision / Cobot 통합
Jetson Orin NX 16GB 비전 서버 구축 (YOLO 기반 불량 감지), vision/1/result 발행 → ESP32 구독, 협동로봇 2대 (RPi5 + ROS2 Jazzy + MoveIt2) — 주조 pouring + pick/place 담당
■ Phase 3 — AMR / ROS2 연동
AMR 3대 (RPi4 + ROS2 Jazzy + Nav2 + SLAM Toolbox), rosbridge_websocket 으로 FastAPI 연동, 작업 할당기 (task_allocator.py) 구현 — 3대 간 부하 분배, 3D 맵에 AMR 실시간 위치 표시 (FactoryMap3D)
■ Phase 4 — 관제 고도화
OEE / KPI 자동 계산 (PostgreSQL Continuous Aggregates 활용), 알람 룰 엔진 (PL/pgSQL 트리거 기반) + 임계값 관리 UI, 이력 조회 / 리포트 생성 (TimescaleDB 압축), MinIO/S3 이미지 저장소, PostgreSQL streaming replication (HA), ERP / MES 외부 연동
