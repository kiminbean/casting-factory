Manhole for SmartCast Robotics

State Diagram Reference — 맨홀 뚜껑(Manhole Cover) 사형 주조 공정 | 8단계 생산 공정 기반

공정 흐름 (8단계)

1. 주문/계획→ 2. 원자재 처리→ 3. 주형 및 주조→ 4. 냉각 및 탈형→ 5. 이송→ 6. 후처리→ 7. 검사→ 8. 분류/출하

상태 정의 (State Definitions)

시작

Idle — 대기

액션내용
Entry시스템 초기화, 센서 셀프 체크
Do주문 수신 대기 (WebSocket 모니터링)
Exit—
1단계

Order_Received — 주문 접수

액션내용
EntryBOM 확인 (제품/수량/자재)
Do생산 오더 생성
Exit자재 요청 발행
2단계
복합 상태 (Composite State)

Raw_Material_Processing — 원자재 처리

Ingot_Loading — 주괴 적재
Entry주괴(Ingot) 적재
Do중량 계측
Exit투입 확인
Melting — 용해
Entry용해로 가열 시작
Do온도 모니터링 (목표 온도까지)
Exit슬래그 제거
3단계
복합 상태 (Composite State)

Mold_And_Casting — 주형 및 주조

Molding — 조형
Entry사형(Sand Mold) 제작 시작
Do코어(Core) 세팅, 주형사 다짐
Exit주형 검사
Pouring — 주탕
Entry래들 이동, 탕구 정렬
Do용탕 주입 (유량/온도 제어)
Exit유량 기록
4단계
복합 상태 (Composite State)

Cooling_And_Shakeout — 냉각 및 탈형

Cooling — 냉각
Entry냉각 타이머 시작
Do온도 센서 모니터링
Exit냉각 완료 로그
Shakeout — 탈형
Entry로봇암(MyCobot280) 호출
Do주형 파쇄, 주물 분리
Exit주물 상태 기록
5단계

Transferring — 이송 (AMR)

액션내용
EntryAMR(Pinkypro) 배정
Do이송 위치 추적 (Nav2)
Exit후처리 구역 도착 확인
6단계

Post_Processing — 후처리 (수작업)

액션내용
Entry작업대 배치
Do버(Burr) 제거, 그라인딩, 표면 청소
Exit완료 후 컨베이어 이송 요청
7단계
복합 상태 (Composite State)

Inspection — 검사

Conveyor_Moving — 컨베이어 이송
Entry컨베이어 벨트 구동
Do이송 중 (ESP32 제어)
Exit—
Vision_Inspecting — 비전 검사
Entry벨트 정지 (TOF250 감지, 0.5초 이내)
DoYOLO + PatchCore 촬영 및 판정
Exit판정 결과 DB 저장
8단계
복합 상태 (Composite State) — 분기

Sorting_And_Shipping — 분류/출하

Good_Palletizing — 양품 적재 [결과 == PASS]
Entry양품 트레이 적재 (로봇암)
Do트레이 배치
Exit출하 대기
Shipping — 출하
EntryAMR 배정
Do출하 구역 이송
Exit주문 상태 → completed ◉
Defect_Disposal — 불량 폐기 [결과 == FAIL]
Entry불량품 폐기함 적재 (로봇암)
Do폐기함 이송
Exit폐기 기록 저장 ◉

State Transition Table (상태 전이 규칙)

#현재 상태이벤트가드다음 상태액션
1Idle주문 접수—Order_ReceivedBOM 확인, 생산 오더 생성
2Order_Received생산 오더 확정—Ingot_Loading자재 요청, 주괴 준비
3Ingot_Loading투입 완료—Melting용해로 가열 시작
4Melting용탕 온도 도달[temp >= 목표 온도]Molding슬래그 제거, 주형 준비 요청
5Molding주형 완성[주형 검사 통과]Pouring래들 이동, 주탕 시작
6Pouring주탕 완료[유량 목표 달성]Cooling유량 기록, 타이머 시작
7Cooling냉각 완료[temp < 안전 온도]Shakeout로봇암(MyCobot280) 호출
8Shakeout탈형 완료—Transferring주물 분리, AMR 배정
9Transferring후처리 구역 도착—Post_Processing작업대 배치
10Post_Processing후처리 완료—Conveyor_Moving컨베이어 적재, 이송 요청
11Conveyor_Moving검사 구간 도착[TOF250 < 임계값]Vision_Inspecting벨트 정지 (0.5초 이내)
12Vision_Inspecting판정 완료 — 양품[결과 == PASS]Good_Palletizing양품 트레이 적재
13Vision_Inspecting판정 완료 — 불량[결과 == FAIL]Defect_Disposal불량품 폐기함 적재
14Good_Palletizing적재 완료—ShippingAMR 배정, 출하 구역 이송
15Shipping출하 완료—◉ 종료주문 상태 → completed
16Defect_Disposal폐기 완료—◉ 종료폐기 기록 저장
