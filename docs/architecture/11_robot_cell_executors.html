① 개요

Robot / Cell Executors는 추상적인 task를 실제 하드웨어 명령으로 변환하여 송출하는 최하위 계층입니다. 각 장비 타입별로 별도의 executor가 존재합니다.

1.1 책임

  • Allocator로부터 assigned task 수신
  • task.payload를 실제 장비 명령으로 변환
  • 장비 프로토콜(ROS2/MQTT/Serial)로 송출
  • 실행 결과를 Execution Monitor에 보고

1.2 범위 밖

  • 의사결정 — Allocator가 이미 결정
  • 경로 계획 — Traffic Manager + Nav2가 담당
  • 재시도 판단 — Execution Monitor가 담당

③ AMR Executor (Pinkypro)

3.1 하드웨어 스펙

  • 크기: 0.12 × 0.12 m
  • 구동: Differential drive (제자리 회전 가능)
  • 컨트롤러: Raspberry Pi 4
  • 미들웨어: ROS 2 Jazzy + Nav2
  • 통신: WiFi + DDS

3.2 주요 Action

# ROS 2 Nav2 action client
from nav2_msgs.action import NavigateToPose
from rclpy.action import ActionClient

class AMRExecutor:
    def __init__(self, node, amr_id: str):
        self.amr_id = amr_id
        self._client = ActionClient(
            node, NavigateToPose,
            f"/{amr_id}/navigate_to_pose"
        )

    async def execute_transport(self, task: Task):
        target = task.payload["destination"]

        goal = NavigateToPose.Goal()
        goal.pose.pose.position.x = target[0]
        goal.pose.pose.position.y = target[1]

        await self._client.wait_for_server()
        future = await self._client.send_goal_async(
            goal, feedback_callback=self._on_feedback
        )

        result = await future.get_result_async()
        if result.status == "SUCCEEDED":
            report_completion(task.task_id, success=True)
        else:
            report_completion(task.task_id, success=False,
                              reason=result.message)

④ Manipulator Executor (MyCobot280)

4.1 하드웨어 스펙

  • 제조사: Elephant Robotics
  • DOF: 6
  • Payload: 250g
  • 컨트롤러: Raspberry Pi 5 (per robot)
  • SDK: pymycobot (Python) + ROS 2 bridge

4.2 작업 타입별 명령

from pymycobot.mycobot import MyCobot
import rclpy

class ManipulatorExecutor:
    def __init__(self, cobot_port: str):
        self.mc = MyCobot(cobot_port, 115200)

    def execute_pouring(self, task: Task):
        # 1. 안전 홈으로 복귀
        self.mc.send_angles([0, 0, 0, 0, 0, 0], 50)

        # 2. 래들 그립 위치로 이동
        ladle_pose = task.payload["ladle_pose"]
        self.mc.send_coords(ladle_pose, 30, 1)
        self.mc.set_gripper_state(1, 50)  # close
        time.sleep(0.5)

        # 3. 주형 입구로 이동
        mold_pose = task.payload["mold_pose"]
        self.mc.send_coords(mold_pose, 30, 1)

        # 4. 기울여 주탕 (설정된 각도 시퀀스)
        for angle, duration in POURING_SEQUENCE:
            self.mc.send_angle(5, angle, 20)  # wrist tilt
            time.sleep(duration)

        # 5. 수평 복귀
        self.mc.send_angle(5, 0, 30)

        # 6. 래들 복귀 + 그립 해제
        self.mc.send_coords(ladle_pose, 30, 1)
        self.mc.set_gripper_state(0, 50)  # open

        # 7. 홈 복귀
        self.mc.send_angles([0, 0, 0, 0, 0, 0], 50)
        report_completion(task.task_id, success=True)

⑤ Station Executor (ESP32 PLC)

컨베이어, 용해로 등은 ESP32 마이크로컨트롤러에서 직접 제어됩니다. Executor는 MQTT를 통해 명령을 전달합니다.

5.1 컨베이어 명령 예시

import json
import paho.mqtt.client as mqtt

class ConveyorExecutor:
    def __init__(self, conveyor_id: int, mqtt_host: str):
        self.id = conveyor_id
        self.client = mqtt.Client(client_id=f"exec_conv_{conveyor_id}")
        self.client.connect(mqtt_host, 1883)
        self.client.loop_start()

        # status 토픽 구독 (실행 완료 판단용)
        self.client.subscribe(f"conveyor/{conveyor_id}/status")
        self.client.on_message = self._on_status

    def execute_run(self, task: Task):
        cmd = {"cmd": "start", "speed": task.payload.get("speed", 80)}
        self.client.publish(
            f"conveyor/{self.id}/cmd",
            json.dumps(cmd)
        )
        # status 토픽 기다림 (asyncio + Future)

    def execute_stop(self, task: Task):
        self.client.publish(
            f"conveyor/{self.id}/cmd",
            json.dumps({"cmd": "stop"})
        )

5.2 ESP32 펌웨어 측 (요약)

실제 펌웨어는 firmware/conveyor_controller/에 구현. v3.4 기준:

  • MQTT 구독: conveyor/{id}/cmd
  • L298N 모터 드라이버 제어 (JGB37-555 6V 모터)
  • TOF250 ×2 센서 (ASCII UART 9600) 기반 상태 머신
  • MQTT publish: conveyor/{id}/status, conveyor/{id}/event

⑥ 공통 인터페이스

모든 Executor는 동일한 기본 인터페이스를 구현:

from abc import ABC, abstractmethod

class Executor(ABC):
    executor_id: str
    capabilities: list[str]

    @abstractmethod
    async def execute(self, task: Task) -> ExecutionResult: ...

    @abstractmethod
    async def cancel(self, task_id: str) -> None: ...

    @abstractmethod
    def health(self) -> HealthStatus: ...

@dataclass
class ExecutionResult:
    task_id: str
    success: bool
    elapsed_sec: float
    output: dict  # 예: inspection 결과, 주탕 각도 기록
    failure_reason: str | None = None

6.1 Executor Registry

class ExecutorRegistry:
    def __init__(self):
        self._executors: dict[str, Executor] = {}

    def register(self, executor: Executor):
        self._executors[executor.executor_id] = executor

    def for_capability(self, cap: str) -> list[Executor]:
        return [
            e for e in self._executors.values()
            if cap in e.capabilities and e.health().available
        ]

    async def dispatch(self, robot_id: str, task: Task):
        executor = self._executors[robot_id]
        result = await executor.execute(task)
        emit_event("task.completed", result=result)

⑦ 구현 스케치

class AMRExecutor(Executor):
    executor_id: str
    capabilities = ["amr"]

    def __init__(self, robot_id: str, node):
        self.executor_id = robot_id
        self.robot_id = robot_id
        self.node = node
        self._nav_client = ActionClient(
            node, NavigateToPose,
            f"/{robot_id}/navigate_to_pose"
        )

    async def execute(self, task: Task) -> ExecutionResult:
        start = time.time()

        try:
            if task.type == TaskType.TRANSPORT:
                await self._navigate(task.payload["destination"])
            elif task.type == TaskType.CHARGING:
                await self._navigate(task.payload["charging_station"])
                await self._dock_and_charge()
            else:
                raise ValueError(f"Unsupported task type: {task.type}")

            return ExecutionResult(
                task_id=task.task_id,
                success=True,
                elapsed_sec=time.time() - start,
                output={"final_position": self._get_position()},
            )
        except Exception as e:
            return ExecutionResult(
                task_id=task.task_id,
                success=False,
                elapsed_sec=time.time() - start,
                output={},
                failure_reason=str(e),
            )

    def health(self) -> HealthStatus:
        return HealthStatus(
            available=self._battery > 15,
            battery=self._battery,
            last_heartbeat=self._last_heartbeat,
        )
