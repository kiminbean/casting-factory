1. 목표 & 동기

Interface Service(FastAPI)가 공장 운영의 SPOF(Single Point of Failure) 였다. Interface 장애·AWS 이관 시에도 공장 가동이 유지되도록 Management Service 를 별도 프로세스(gRPC :50051) 로 분리하고, Factory Operator PC(PyQt)가 직접 gRPC 로 호출하게 한다.

[ Factory Operator PyQt ] ──gRPC:50051──▶ [ Management Service ] │ │ PostgreSQL 공유 ▼ [ Admin / Customer Web ] ──HTTP──▶ [ Interface Service (FastAPI :8000) ]
