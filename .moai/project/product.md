# 주물공장 생산 관제 시스템 (Casting Factory MES)

## 프로젝트 개요

주물(캐스팅) 스마트 공장의 실시간 생산 관제 시스템. 용해-주형제작-주탕-냉각-탈형-후처리-검사-분류의 전체 주물 생산 공정을 모니터링하고, 주문 관리, 품질 검사, 물류/이송, 설비 상태를 통합 대시보드로 관리한다.

## 대상 사용자

- 공장 관리자 (생산 현황 모니터링, KPI 추적)
- 생산 담당자 (공정별 상태 확인, 알림 대응)
- 품질 관리자 (검사 결과 조회, 불량률 분석)
- 물류 담당자 (AMR 이송 현황, 창고 관리)

## 핵심 기능

### 1. 통합 대시보드 (Dashboard)
- 생산 목표 달성률, 가동 로봇 수, 미처리 주문, 알람 요약 카드
- 2D 공장 레이아웃 맵 (설비 위치/상태 시각화)
- 실시간 알림 피드 (심각도 순 정렬)
- 주간 생산 추이 차트 (Recharts)
- 최근 주문 테이블

### 2. 3D 공장 맵 (Factory Map 3D)
- Three.js 기반 실시간 3D 공장 뷰어
- Blender에서 제작한 GLB 모델 렌더링
- 설비 배치 편집 모드 (Ctrl+D 복사, Delete 삭제)
- AMR 복도 전용 경로 시각화
- 카메라 180도 회전, 스페이스 Pan 지원

### 3. 주문 관리 (Order Management)
- 주문 접수/검토/승인/생산/출하/완료 상태 추적
- 고객사 정보, 납기일, 금액 관리
- 상세 품목 (규격, 재질, 후처리, 로고 데이터)

### 4. 생산 모니터링 (Production Monitoring)
- 8단계 공정 상태 추적 (용해~분류)
- 공정별 온도, 압력, 출력, 냉각 진행률 실시간 데이터
- 설비별 가동 시간, 에러 카운트

### 5. 품질 검사 (Quality Control)
- 비전 시스템 기반 자동 검사 결과
- 합격/불량 판정, 불량 유형 분류
- 불량률 차트 및 불량 유형 분포 시각화

### 6. 물류 및 이송 (Logistics & Fleet)
- AMR 이송 태스크 관리 (배정~완료)
- 창고 랙 상태 (빈/점유/예약)
- 출고 오더 (FIFO/LIFO 정책)

### 7. 실시간 통신
- WebSocket을 통한 설비 상태/알림 실시간 갱신

## 비즈니스 도메인

- 산업: 주물(캐스팅) 제조업
- 시스템 유형: MES (Manufacturing Execution System) / SCADA 대시보드
- 공정 특화: 용해로 온도 관리, 주형 압력 제어, 냉각 프로세스, 비전 검사

## 현재 상태

- **프론트엔드**: 대시보드 + 3D 맵 뷰어 구현 완료, Mock 데이터 기반
- **백엔드**: FastAPI REST API + WebSocket 구현 완료, SQLite 시드 데이터
- **3D**: Blender GLB 모델 + Three.js 뷰어/편집기 구현 완료
- **테스트**: 미작성 (0개)
- **백엔드 연결**: 프론트엔드는 아직 Mock 데이터 사용 중 (API 연동 대기)
