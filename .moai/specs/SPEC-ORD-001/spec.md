# SPEC-ORD-001: 고객 발주 페이지 개선 (SR-ORD-01-01 ~ 03)

## HISTORY

| 날짜 | 버전 | 변경 내용 |
|------|------|----------|
| 2026-04-02 | 1.0.0 | 초기 SPEC 생성 |

## 개요

고객 발주 페이지(`/customer`)의 제품 조회, 옵션 선택, 디자인 확인 기능을 개선한다.

## 요구사항 (EARS Format)

### R1: 카테고리 필터링 [SR-ORD-01-01]
사용자가 제품 목록 상단에서 카테고리(맨홀뚜껑, 그레이팅 등)를 선택하면, 해당 카테고리의 제품만 표시되어야 한다.

### R2: 제품 상세 정보 표시 [SR-ORD-01-01]
제품 카드에 이름, 규격, 가격 외에 재질 목록과 하중 등급 범위가 함께 표시되어야 한다.

### R3: 이미지 카드형 후처리 선택 [SR-ORD-01-02]
후처리 옵션(표면 연마, 방청 코팅, 아연 도금, 로고 삽입)이 체크박스 대신 이미지 카드 형식으로 제공되어야 한다.

### R4: 디자인 시안 표시 [SR-ORD-01-03]
Step 3(견적 확인)에서 선택한 제품의 기본 디자인 시안 이미지가 표시되어야 한다.

### R5: 옵션 선택 미리보기 [SR-ORD-01-03]
Step 3에서 선택한 규격/재질/후처리 옵션이 시각적 요약 카드 형태로 표시되어야 한다.

## 범위

### 포함
- `/customer` 페이지(src/app/customer/page.tsx) 수정
- 카테고리 필터 UI
- 제품 카드 재질/하중등급 추가
- 후처리 이미지 카드 UI
- 견적 확인 단계 디자인 시안 + 시각적 미리보기

### 제외
- 실제 제품 이미지 파일 (placeholder 아이콘 유지)
- 백엔드 API 연동 (프론트엔드 로컬 데이터 기반)
- 3D 제품 미리보기
